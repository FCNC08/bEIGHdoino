beighduino

beighduino/beighduino shield sind Arduino Bibilotheken, um eigens bei beight(https://github.com/FCNC08/beight) auf einem Arduino zu nuzten. 
Für beighduino shield ist ein dafür speziell entwickeltes Shield für einen Arduino Uno nötig.
